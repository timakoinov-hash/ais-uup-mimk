ОБЩЕЕ ОПИСАНИЕ ПРЕДМЕТНОЙ ОБЛАСТИ
=======================================

Проект: Автоматизированная информационная система 
«Учёт успеваемости и посещаемости студентов ГБПОУ «МиМК»»
(сокращённо — АИС УУП)

1. ОРГАНИЗАЦИЯ-ЗАКАЗЧИК
------------------------
ГБПОУ «Миасский машиностроительный колледж»
Адрес: 456300, Челябинская обл., г. Миасс, пр. Октября, д. 1
Тел.: +7 (3513) 55-44-33

2. ОБЪЕКТ АВТОМАТИЗАЦИИ
------------------------
Процесс учёта успеваемости и посещаемости студентов всех форм обучения.

3. ОСНОВНЫЕ СУЩНОСТИ
---------------------
- Студент (ФИО, группа, контакты)
- Группа (наименование, специальность, курс)
- Преподаватель (ФИО, должность, отделение)
- Дисциплина (наименование, специальность, семестр)
- Занятие (дисциплина, преподаватель, группа, дата, время)
- Журнал (занятие, студент, оценка, посещение, причина пропуска)
- Пользователь (логин, пароль, роль)

4. ОСНОВНЫЕ РОЛИ ПОЛЬЗОВАТЕЛЕЙ
-------------------------------
- Администратор — полный доступ
- Заведующий отделением — контроль, отчёты
- Преподаватель — ведение журнала
- Куратор — контроль группы, уведомления
- Студент — просмотр оценок
- Родитель — просмотр оценок ребёнка

5. КЛЮЧЕВЫЕ ФУНКЦИИ
--------------------
- Ведение электронного журнала
- Автоматический расчёт успеваемости и посещаемости
- Генерация отчётов (Excel, PDF)
- Личный кабинет студента/родителя
- Автоматические уведомления о пропусках
- Экспорт данных

6. ТЕХНОЛОГИЧЕСКИЙ СТЕК
------------------------
- Язык программирования: Python (Django)
- База данных: PostgreSQL
- Фронтенд: React.js
- Контроль версий: Git (GitHub)
- Сервер: Ubuntu 20.04 LTS
