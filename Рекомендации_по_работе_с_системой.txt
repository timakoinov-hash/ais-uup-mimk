РЕКОМЕНДАЦИИ ПО РАБОТЕ С СИСТЕМОЙ УЧЁТА УСПЕВАЕМОСТИ
=======================================================

1. ДЛЯ ПРЕПОДАВАТЕЛЕЙ
----------------------
1.1. Заполняйте электронный журнал сразу после занятия — это исключает ошибки.
1.2. При выставлении оценки указывайте комментарий (при необходимости).
1.3. Отмечайте причины пропусков (уважительная/неуважительная) корректно.
1.4. Используйте функцию «Быстрый ввод» для экономии времени.
1.5. Периодически проверяйте автоматический расчёт среднего балла.
1.6. При обнаружении ошибки — внесите корректировку через специальную форму.

2. ДЛЯ КУРАТОРОВ ГРУПП
----------------------
2.1. Еженедельно просматривайте сводку по успеваемости группы.
2.2. При выявлении систематических пропусков — инициируйте отправку уведомления родителям.
2.3. Используйте систему для формирования справок и характеристик.
2.4. Контролируйте своевременность заполнения журналов преподавателями.
2.5. Проводите анализ успеваемости в разрезе дисциплин и отдельных студентов.

3. ДЛЯ СТУДЕНТОВ
-----------------
3.1. Регулярно проверяйте свой личный кабинет для контроля успеваемости.
3.2. При обнаружении ошибки в оценке — обращайтесь к преподавателю.
3.3. Используйте данные системы для планирования подготовки к экзаменам.
3.4. Следите за уведомлениями о задолженностях.

4. ДЛЯ РОДИТЕЛЕЙ
-----------------
4.1. Зарегистрируйтесь в системе, используя код доступа, полученный от куратора.
4.2. Регулярно просматривайте успеваемость и посещаемость ребёнка.
4.3. При снижении успеваемости — свяжитесь с куратором через систему.
4.4. Используйте мобильное приложение для удобного доступа.

5. ДЛЯ АДМИНИСТРАТОРА СИСТЕМЫ
------------------------------
5.1. Ежедневно проверяйте работу системы и состояние резервных копий.
5.2. Обновляйте справочники (группы, дисциплины, преподаватели) в начале каждого семестра.
5.3. Управляйте пользователями и правами доступа.
5.4. Мониторьте журнал событий (логи) на предмет подозрительных действий.
5.5. Периодически проводите обучение пользователей работе с системой.

6. ОБЩИЕ РЕКОМЕНДАЦИИ
---------------------
6.1. Используйте только официальные версии системы (не тестовые).
6.2. Не передавайте свои учётные данные третьим лицам.
6.3. При возникновении технических проблем — обращайтесь в службу поддержки.
6.4. Регулярно обновляйте программное обеспечение (при наличии обновлений).
6.5. Соблюдайте требования информационной безопасности (сложные пароли, двухфакторная аутентификация).
