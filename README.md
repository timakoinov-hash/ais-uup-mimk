# АИС «Учёт успеваемости и посещаемости студентов ГБПОУ «МиМК»»

## Описание проекта
Автоматизированная информационная система для учёта успеваемости и посещаемости студентов 
Государственного бюджетного профессионального образовательного учреждения 
«Миасский машиностроительный колледж».

## Цель проекта
Создание централизованной системы, позволяющей фиксировать, хранить и обрабатывать 
данные об успеваемости и посещаемости студентов с минимальным участием человека 
и высоким уровнем достоверности.

## Основные функции
- Ведение электронного журнала успеваемости
- Автоматический расчёт среднего балла и процента посещаемости
- Генерация отчётов по группам и специальностям
- Личный кабинет студента и родителя
- Автоматическое уведомление о пропусках и задолженностях

## Технологический стек
- Backend: Python / Django
- Frontend: React / Vue.js
- База данных: PostgreSQL
- Контроль версий: Git / GitHub
